#### 1️⃣ 진행한 팀 활동
- **주요 활동:** 교수님 및 대학원생 선배님과의 인터뷰 진행
  - **목표:** 조원 6명과 교수님과 겹치는 시간을 찾아 인터뷰를 진행
  - **도구:** [When2meet](https://www.when2meet.com/) 사용하여 시간 조율

#### 2️⃣ 팀 활동 방향
- **활동 선정 기준:** 팀 발표 활동 시 조사한 관심사를 기반으로 설정
  - **분야:** 정보보호, 게임 프로그래밍, 인공지능 등
